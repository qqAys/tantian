# 隐私政策

**生效日期：** 2025-11-26

欢迎使用谈天（以下简称“本应用”）。本隐私政策旨在说明本应用在您使用服务过程中，如何收集、使用和保护您的信息。

## 1. 信息的收集与使用

### 1.1 自动生成的标识符 (User ID)

* **收集的信息：** 当您首次访问本应用时，系统会为您自动生成一个**唯一的、匿名的用户ID**（基于 `uuid4`）。
* **存储位置：** 这个ID存储在您的**浏览器本地存储** 中。
* **使用目的：** 用于在您后续访问时识别您是同一用户，并用于生成您的**随机头像**（通过 `robohash.org`）以及作为您发送消息时的**身份标识**。

### 1.2 聊天内容

* **收集的信息：** 您发送的**聊天文本**和发送的**时间戳**。
* **使用方式：** 这些信息被用于**实时展示**给所有在线用户。
* **存储政策：** **本应用不会将您的历史聊天消息持久化存储到任何数据库或文件中。**
    * 消息仅在程序**运行时**保存在服务器的**内存列表**中。
    * 一旦服务器重启或程序终止，所有聊天记录将**被清除**。

### 1.3 在线状态

* **收集的信息：** 您的**用户ID**会被添加至服务器内存中的**在线用户列表**。
* **使用目的：** 用于**实时统计并显示**当前在线的用户数量。当您断开连接时，您的ID会从列表中移除。

## 2. 信息存储与数据安全

* **存储位置：** 您的用户ID存储在**您的浏览器**中。聊天消息仅存储在**本应用服务器的运行内存**中，不进行持久化存储。
* **数据安全：** 由于本应用**不存储**用户的个人身份信息（如姓名、电话、邮箱）和历史消息，因此泄露敏感信息的风险极低。
* **定期清除：** 为确保数据隐私和系统性能，本应用会在**UTC时间每日 00:00**（午夜）自动执行内存数据清除操作。届时，所有实时聊天记录和当前在线用户状态将被**清空**。
